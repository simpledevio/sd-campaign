What makes Simple Dev different

Short, easy-to-read tutorials: All of the tutorials on Simple Dev are written and follow a standard format that is short, easy-to-read tutorials.

Cross platform: Tutorials include instructions for macOS and Windows users where applicable.

Local development: Learn how to set up your computer for development. Learn how to install a text editor, Git, npm, Sass, and more.

Transferable skills: Learn skills that go beyond web development, like the terminal, Git, GitHub, Markdown, and more.
